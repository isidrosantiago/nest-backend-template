<h1 align="center">NESTJS BACKEND TEMPLATE</h1>
  
### Description
Nestjs backend template with authentication using jsonwebtoken and postgres

## Project setup

```bash
$ npm install
```

Update connection settings `src/ormdatasource.ts`

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# drop database
$ npm run db:drop

# create migration
$ npm run db:create

# run migration
$ npm run db:migrate
```
